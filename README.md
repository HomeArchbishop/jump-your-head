## 📦️ 应用部署

### 安装依赖 (Install dependencies)

```
npm install
```

### 打包 (Compiles and minifies for production)
```
npm run build
```

### 启动 (Start with pm2)

```
npm start
# 默认端口7777
```

### 停止 (Stop with pm2, actually delete the pm2 app)

```
npm stop
```



## 🔨 开发

### 安装依赖 (Install dependencies)

```
npm install
```

### 热重载启动 (Compiles and hot-reloads for development)

```
npm run serve
```

### Lints and fixes files

```
npm run lint
```
